My Simple Blog (Учебный проект)
Это демонстрационный проект «Простой блог», созданный исключительно с использованием HTML. Цель проекта — отработать навыки структурирования веб-страниц, использования семантических тегов и работы с гиперссылками.

 Цели проекта
Изучение базовой структуры HTML-документа.

Работа с текстовыми тегами (<h1>-<h6>, <p>, <ul>).

Создание навигации между страницами через тег <a>.

Размещение мультимедиа контента (изображения).

Практика использования семантической разметки (<header>, <main>, <article>, <footer>).

 Структура файлов
Проект состоит из следующих страниц:

index.html — главная страница со списком всех постов.

about.html — страница об авторе блога.

post-1.html, post-2.html — примеры страниц с полным текстом статей.

 Функционал
Главная страница: Содержит приветствие, краткое описание блога и список анонсов статей.

Навигация: Меню, которое позволяет переходить между главной, страницей "О себе" и самими постами.

Статьи: Каждая статья оформлена с заголовком, датой публикации, картинкой и основным текстом.

Подвал (Footer): Содержит контактную информацию и ссылки на социальные сети (заглушки).

 Как запустить
Скачайте все файлы репозитория.

Убедитесь, что все файлы находятся в одной папке.

Откройте файл index.html в любом современном браузере (Chrome, Firefox, Safari, Edge).
